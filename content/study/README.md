# Study notes folder

Put one markdown file per topic in this folder.

- The Study page will list all `*.md` files here.
- It will automatically use the **first image** in each markdown file as the thumbnail.
- If a markdown file does not include an image, the page will use `/study-assets/diffusion/test1.png` as the default thumbnail.

Example:

```md
# Diffusion Models

![thumbnail](/study-assets/diffusion/test1.png)

Your notes...
```
